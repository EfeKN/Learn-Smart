# Build

Use node 20.15.0

Use python 3.12.4

# Frontend

`npm install`

`npm run dev`

# Backend

`python -m venv backend`

`backend\Scripts\activate`

`cd backend`

`pip install -r requirements.txt`

`uvicorn main:app --reload`
